README for data_analysis_project:

Download the whole folder to run analysis in visual studio code or your IDE of choice. The scripts have been tested and works across Windows and Mac OS platforms. 

The resulting interactive dashboard should launch locally in the browser. Failing this, screenshots of the plots are appended in the project summary document. 

In the interactive dashboard,

to view the plots, go to the year slider and slide it all the way to 2016. This will display all data form 1975 to 2016. 

Below, you will find a menu that filters the data according to sex (i.e. male and female).
